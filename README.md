## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Prashan608/Recipe-sharing-react.git
    cd Recipe-sharing-react/recipe-sharing-platform
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start local development server:
    ```bash
    npm run dev
    ```

4. Open the browser and visit:
    ```
    http://localhost:5173
    ```
